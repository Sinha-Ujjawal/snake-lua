# Snake Game in Love2d
This is a snake game created in [love2d](https://love2d.org/).

![game.png](./game.png)

## Getting Started
1. Download [love2d](https://love2d.org/) executable
2. `cd` to this cloned repo and run the game-
```console
love .
```

## Controls
Arrows to control the snake
